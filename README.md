# Setup:
- `uv sync`
- `cd /absolute/path/to/weave-python/weave-public/weave && git checkout 'zach/realtime-ga' && cd -`
- `ln -s /absolute/path/to/weave-python/weave-public/weave $(pwd)/weave`

## Usage:
 - websockets example: `uv run websockets_example.py`
 - agents-sdk example: `uv run agents_sdk_example.py`
